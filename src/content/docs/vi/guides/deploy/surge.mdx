Bạn có thể triển khai dự án Astro của mình lên [Surge](https://surge.sh/), một nền tảng xuất bản web chỉ với một lệnh, được thiết kế cho các lập trình viên front-end.

## Cách triển khai

<Steps>
1. Cài đặt [Surge CLI](https://www.npmjs.com/package/surge) toàn cục từ terminal, nếu bạn chưa cài.

    ```shell
    npm install -g surge
    ```

2. Xây dựng trang Astro của bạn từ thư mục gốc của dự án.

    ```shell
    npm run build
    ```

3. Triển khai lên Surge bằng CLI.

    ```shell
    surge dist
    ```

    Bạn có thể [sử dụng tên miền tùy chỉnh với Surge](http://surge.sh/help/adding-a-custom-domain) khi triển khai bằng cách chạy `surge dist yourdomain.com`.
</Steps>
