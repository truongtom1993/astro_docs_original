Bây giờ bạn đã có cùng một mã HTML xuất hiện ở nhiều trang trên trang web Astro của mình, đã đến lúc thay thế nội dung bị lặp lại đó bằng một component Astro có thể tái sử dụng!

## Tạo thư mục `src/components/` mới

Để lưu các tệp `.astro` sẽ sinh HTML nhưng không trở thành các trang mới trên website của bạn, bạn cần một thư mục mới trong dự án: `src/components/`. 


## Tạo component Điều hướng

<Steps>
1. Tạo một tệp mới: `src/components/Navigation.astro`.

2. Sao chép các liên kết điều hướng giữa các trang từ đầu bất kỳ trang nào và dán chúng vào tệp mới `Navigation.astro`:

    ```astro title="src/components/Navigation.astro" "---"
    ---
    ---
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    ```
    :::tip
    Nếu không có gì trong frontmatter của tệp `.astro` của bạn, bạn không cần phải viết các dấu code fences. Bạn luôn có thể thêm chúng lại khi cần.
    :::
</Steps>

### Import và sử dụng Navigation.astro

<Steps>
1. Quay lại `index.astro` và import component mới của bạn vào bên trong phần code fence:

    ```astro title="src/pages/index.astro" ins={2}
    ---
    import Navigation from '../components/Navigation.astro';
    import "../styles/global.css";

    const pageTitle = "Home Page";
    ---
    ```

2. Sau đó, bên dưới, thay thế các phần tử HTML liên kết điều hướng hiện tại bằng component điều hướng mà bạn vừa import:

    ```astro title="src/pages/index.astro" del={1-3} ins={4}
    <a href="/">Home</a>
    <a href="/about/">About</a>
    <a href="/blog/">Blog</a>
    <Navigation />
    ```

3. Xem trước trên trình duyệt và để ý rằng nó sẽ trông giống hệt như trước... và đó là điều bạn mong muốn! 
</Steps>

Trang web của bạn chứa cùng một mã HTML như trước. Nhưng bây giờ, ba dòng mã đó được cung cấp bởi component `<Navigation />`. 

<Box icon="puzzle-piece">

## Thực hành - Thêm điều hướng cho các trang còn lại

Hãy import và sử dụng component `<Navigation />` trong hai trang còn lại của bạn (`about.astro` và `blog.astro`) theo cách tương tự.

Đừng quên
- Thêm câu lệnh import ở đầu script của component, bên trong code fence.
- Thay thế mã cũ bằng component điều hướng.

</Box>

:::note
Khi bạn cấu trúc lại mã nguồn nhưng không thay đổi giao diện trang trên trình duyệt, bạn đang **refactor** (tái cấu trúc). Bạn sẽ **refactor** nhiều lần trong phần này khi thay thế các phần của mã HTML bằng các component.

Điều này cho phép bạn bắt đầu nhanh chóng với mã nguồn hoạt động, thường bị lặp lại trong dự án. Sau đó, bạn có thể cải thiện thiết kế mã hiện tại từng bước mà không làm thay đổi giao diện bên ngoài của trang web.
:::
