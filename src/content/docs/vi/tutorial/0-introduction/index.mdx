<Lede> Trong hướng dẫn này, bạn sẽ học các tính năng chính của Astro bằng cách xây dựng một blog hoàn chỉnh, từ con số 0 đến khi ra mắt! 🚀 </Lede>

Trong quá trình học, bạn sẽ: 
- Thiết lập môi trường phát triển
- Tạo các trang và bài viết blog cho trang web của mình
- Xây dựng với các thành phần (component) của Astro
- Truy vấn và làm việc với các tệp cục bộ
- Thêm tính tương tác cho trang web
- Triển khai trang web của bạn lên Internet

Muốn xem trước những gì bạn sẽ xây dựng? Bạn có thể xem dự án hoàn chỉnh trên [GitHub](https://github.com/withastro/blog-tutorial-demo) hoặc mở phiên bản hoạt động trong môi trường lập trình trực tuyến như [IDX](https://idx.google.com/import?url=https:%2F%2Fgithub.com%2Fwithastro%2Fblog-tutorial-demo%2F) hoặc [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro).


:::note
Nếu bạn muốn khám phá Astro với một trang web mẫu có sẵn, hãy truy cập https://astro.new và chọn một mẫu khởi đầu để mở và chỉnh sửa trực tuyến.
:::
